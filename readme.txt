This is Our Custom Preloader plugin in wordpress.
Custom Preloader in wordpress is a jQuery Custome Preloader plugin for wordpress. This plugin will enable Custome Preloader when loading your site. By installing this plugin your site will show a Custome Preloader.

== Description ==

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/custom-preloader` directory, or install the plugin through the WordPress Custom Preloader screen directly.
2. Activate the plugin through the 'Custom Preloader' screen in WordPress
3. Use the Settings->Preloader Settings screen to configure the plugin
